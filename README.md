# HACCC
 Homogenized Analysis of Cross-Country Climate Policy

Author: Leonard Missbach (leonard.missbach@pik-potsdam.de)

This repository contains code to reproduce the Homogenised Analysis of Cross-Country Consumption Data with a focus on Carbon Pricing.

Repository contains data to the maximum extent possible. Note that underlying microdata is available upon reasonable request from the authors and with permission from each statistical authority.

This respository also contains matching tables for each country and code written for each country-level analysis.
